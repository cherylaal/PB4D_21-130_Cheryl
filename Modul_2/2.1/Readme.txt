Aplikasi ini berisi data mahasiswa Universitas Trunojoyo Madura.
Dalam pembuatannya, aplikasi ini dibuat dengan beberapa jenis layout.
Yang pertama ada Frame Layout yang digunakan supaya gambar dapat ditumpuk dengan komponen lain atau dapat disebut sebagai background.
Kemudian ada Linear Layout yang digunakan untuk menata tampilan secara keseluruhan dan juga pada bagian judul.
Table Layout digunakan pada bagian data mahasiswa agar terlihat lebih tertata dan rapi.
Yang terakhir, untuk bagian Edit text dan button diatur menggunakan Relative Layout.
