Aplikasi ini berisi daftar beberapa Universitas yang ada di Indonesia.
Aplikasi ini dibuat dengan menggunakan RecycleView. 
RecycleView memudahkan untuk menampilkan kumpulan data dalam jumlah yang besar 
secara efisien, lebih modern dari ListView.
Untuk membuat project RecycleView, diperlukan adanya beberapa file lain sebagai pendukung dalam membangun project tersebut seperti ViewHolder dan Adapter.
