Aplikasi ini merupakan aplikasi untuk mengkonversi file pdf dan 
sebaliknya. Pada halaman pertama terdapat 3 button, yaitu button about, 
website, dan try application. Button about menerapkan intent explicit, jadi 
ketika button about diklik maka user akan diarahkan ke activity lain. 
Sedangkan button website dan try application menerapkan intent implicit. 
Ketika button website, maka user akan diarahkan ke halaman web sesuai 
dengan url yang ada pada „MainActivity.kt‟ yaitu website ilovepdf. Ketika 
button my application diklik maka user akan diarahkan ke aplikasi 
ilovepdf, sedangkan jika user belum memiliki aplikasi maka akan tampil 
pop-up App Market perintah untuk download aplikasi ilovepdf
